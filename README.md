# Slab Theme

## Install

```
git clone https://github.com/chengyin/slab-syntax-atom.git
ln -s ${PWD}/slab-syntax-atom ${HOME}/.atom/packages/slab-syntax
```

## UI Integration

This theme best integrates with Atom's [One Dark UI](https://atom.io/themes/one-dark-ui).
